# Monty Hall — interactive demo

A play-live Monty Hall game for the Monash DeepNeuron "Plan Like an Engineer"
training talk (the *Validate before you build* section). Pick a real door, watch
it swing open, stay or switch, and watch it converge to **stay 33% / switch 67%**.

**Live:** https://aaroneliaszachariah.github.io/monty-hall-demo/

Self-contained single file (`index.html`) — no build, no dependencies. Embed the
live URL in Canva/PowerPoint, or link out to it during the talk.
